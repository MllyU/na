###注意
* 商品价值标志 1-10档(*未想好名称*)
*  商品需要审核，填写商品价值，和交换预估偏差值
*  可向商品发起申请，商品价值，与交换商品价值的差不可大于偏差值
*   商品可选择公开，与不公开。将它申请与其他商品交换后，会自动设为不公开
*  有交换申请的商品不能去交换其他商品。必须先处理**已有申请**
*  确认交易由被交换人填写交易地点，时间。
*  交易完成。互相评价。交易失败，选择取消交易。填写原因。可举报，可申诉
*  评价信息在双方个人中心可见。
